**Заметки, идеи, предложения в свободной форме**
папка core хранит в себе основную логику приложения. Основную бизнес логику, в ней может быть обработка данных,
обработка данных, обработчик ошибок, работа с различными сервисами.
папка rest хранит в себе все, что взаимодействует с внешним миром. Контроллеры, запросы, которые уходят в core и от него
приходят ответы.

Когда пользователь отправляет POST-запрос с JSON в теле, метод calculatePremium будет вызван.

Контроллер извлекает данные из тела запроса, преобразует их в объект TravelCalculatePremiumRequest.

Затем он передает этот объект в сервис TravelCalculatePremiumService для выполнения бизнес-логики (в данном случае расчета премии).

После этого контроллер возвращает результат вычисления в виде объекта TravelCalculatePremiumResponse,
который также будет преобразован в JSON и отправлен клиенту в ответ.
