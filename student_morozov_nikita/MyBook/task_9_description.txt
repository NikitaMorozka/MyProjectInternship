 repositories{} - говорит откуда будет подгружать артефакты (пакеты, плагины, библиотеки)
В Gradle плагины являются модулями, которые расширяют функциональность сборочного процесса.
Они добавляют конкретные задачи, настройки и зависимости, которые упрощают работу с проектом.

Основное назначение плагинов — автоматизация типичных действий,
таких как компиляция, тестирование, упаковка, управление зависимостями и деплой, расширение сборочной системы.

в плагине java происходит автоматическая настройка java-компилятора
в плагине idea происходит интеграция проекта для среды разработки, генерирует файлы
конфигурации (.iml, .ipr, .iws), чтобы облегчить импорт gradle-проекта в IDE.
